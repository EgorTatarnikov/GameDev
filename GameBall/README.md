# <img src="https://raw.githubusercontent.com/EgorTatarnikov/GameDev/main/GameBall/Assets/Sprites/GameBall_r205.svg" width="30" height="30" alt="wt badge"> Game Ball

**[Скачать игру в RuStore](https://www.rustore.ru/catalog/app/com.EgorTatarnikov.GameBall)**

Мобильная игра, где нужно управлять шариком, наклоняя смартфон, чтобы собирать маленькие шарики и зарабатывать очки.<br>
Простое управление с помощью акселерометра делает игру доступной для всех, от детей до взрослых.

Игра демонстрирует мои навыки в разработке игр на Unity. Все элементы игры разработаны мной самостоятельно - графика, музыка и код.

В репозитории представлены только файлы исходного кода; ассеты, готовая сборка и другие файлы отсутствуют.
